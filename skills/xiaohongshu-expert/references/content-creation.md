## 一、选题策划方法论

### 选题的三个来源

**1. 搜索热词挖掘**
- 在小红书搜索框输入核心关键词，查看下拉联想词
- 这些联想词 = 用户真实搜索需求 = 你的选题方向
- 工具：小红书搜索框、千瓜数据、新红数据

**2. 爆款笔记拆解**
- 找同赛道近30天点赞超1000的笔记
- 分析：标题结构、选题角度、内容形式
- 不是抄袭，是拆解底层逻辑后用自己的方式重新表达

**3. 用户痛点收集**
- 翻评论区：用户在问什么问题？
- 翻私信：粉丝最常问你什么？
- 翻竞品：竞品评论区有哪些未满足的需求？

### 选题筛选标准

一个好选题必须满足：
1. **搜索量大**：有人在搜这个话题
2. **你有话说**：你能提供有价值的内容
3. **能引发互动**：用户看完想点赞/收藏/评论
4. **与定位匹配**：不跑偏你的账号方向

### 选题储备建议

- 建一个选题库（Excel/备忘录/Notion都行）
- 每周至少储备10个选题
- 按"干货类/种草类/互动类/蹭热点类"分类管理

## 二、标题写作

### 标题的生死线

标题前10个字权重占比超60%。标题前18字必须包含2个核心关键词。用户刷到你的笔记，只给标题0.5秒的判断时间。

### 标题四大原则

1. **引人注意**：制造好奇、反差、紧迫感
2. **突出价值**：一眼看出"看完我能得到什么"
3. **量化表达**：数字比文字更抓眼球
4. **突出差异**：和同类标题拉开距离

### 标题长度

20字以内最佳。太长在信息流中会被截断。

### 标题公式模板库

**数字型**：
- 「5个方法让你的皮肤白2个度」
- 「月薪3000到月入3万，我做对了这3件事」
- 「99%的人不知道的10个穿搭技巧」

**问题型**：
- 「为什么你总是存不下钱？」
- 「换季皮肤干燥脱皮怎么办？」
- 「25岁该用什么护肤品？」

**反差型**：
- 「不化妆也被夸好看的秘密」
- 「零基础30天学会视频剪辑」
- 「月薪5000也能穿出高级感」

**痛点型**：
- 「毛孔粗大？这个方法亲测有效」
- 「拍照总是不好看？学会这3个角度」
- 「减肥总反弹的人一定要看」

**热点型**：
- 「2025最火的xx，你试过了吗？」
- 「全网都在用的xx，真的好用吗？」

**测评型**：
- 「xx vs xx，到底选哪个？」
- 「花了3000块测了20款面霜，最好用的是这5款」

**教程型**：
- 「手把手教你xx」
- 「保姆级教程：从零学会xx」

### 标题SEO优化

- 核心关键词放标题前10个字
- 前18字包含2个搜索词
- 使用用户真实搜索的词汇（从搜索联想词获取）
- 避免生僻词和行话

## 三、封面设计指南

### 封面的决定性作用

在"发现"页面，用户先看封面再看标题。封面决定了用户是否停下来看你。

### 封面设计公式

**痛点视觉化 + 弱干扰背景 + 情绪符号强化**

### 封面设计规范

- **尺寸**：3:4竖版构图（1080×1440px）
- **主体突出**：人物/产品占画面60%+
- **文字大号**：信息流中缩略图也能看清
- **配色统一**：建立账号视觉风格
- **信息密度**：多宫格拼图适合干货类内容

### 不同类型的封面策略

**干货教程类**：
- 大字标题 + 简洁背景
- 列出核心知识点
- 参考"PPT风格"设计

**种草测评类**：
- 产品实拍为主
- 对比图/效果图
- 标注关键信息（价格/品牌）

**日常分享类**：
- 真人出镜/场景照
- 自然光拍摄
- 氛围感营造

**攻略合集类**：
- 多宫格拼图
- 编号排列
- 信息密度要高

### 封面制作工具

- 醒图/美图秀秀：基础修图
- Canva：模板设计
- 稿定设计：小红书专用模板
- 剪映：视频封面截取

## 四、正文结构框架

### 正文基本结构

**开头（引起兴趣）**：
- 痛点引入："你是不是也有这个烦恼？"
- 结果前置："用了这个方法，我xx天达到了xx效果"
- 反常识开头："90%的人都做错了这一步"

**中间（提供价值）**：
- 逐步展开你的方法/经验/推荐
- 用编号/分点，方便阅读
- 每个要点配合具体例子或数据
- 短段落，每段不超过3行

**结尾（引发互动）**：
- 总结核心观点
- 设置互动问题："你们觉得哪个更好？"
- 引导收藏："建议收藏慢慢看"
- 引导关注："关注我获取更多xx干货"

### 正文排版规范

- 善用emoji作为分段标记（⭐✅💡📌等）
- 每段不超过3行
- 关键信息用【】或「」框起来
- 数字用阿拉伯数字，更醒目
- 适当空行，增加可读性
- 正文字数：图文笔记建议300-800字

### 正文SEO关键词布局

- 前200字自然嵌入长尾关键词
- 每300字重复1次目标关键词
- 不要堆砌关键词，自然融入语境
- 在内容结尾处再次出现核心关键词

## 五、图文笔记创作

### 6种图文形式

1. **实拍产品/场景图**：最真实，适合种草
2. **真人出镜图**：建立信任感，适合人设型账号
3. **海报/PPT图**：适合干货知识类内容
4. **截图图**：适合信息整理类内容
5. **拼图**：多信息展示，适合合集类
6. **手写笔记图**：有温度感，适合阅读/学习类

### 图文笔记最佳实践

- 图片数量：6-9张为最佳
- 第一张图 = 封面，决定点击率
- 每张图承载一个信息点
- 图片之间要有逻辑递进关系
- 最后一张图可做互动引导或个人介绍

## 六、视频笔记创作

### 视频类型

1. **真人出镜口播**：信任感最强，适合知识/经验分享
2. **不出镜配音**：门槛最低，适合不想露脸的创作者
3. **混合模式**：真人+素材+字幕，信息密度高
4. **素材剪辑+配音**：适合盘点/合集类内容

### 视频最佳实践

- **时长**：1-3分钟最佳（完播率是关键指标）
- **开头3秒**：必须抓住注意力，直接说结论或抛出痛点
- **画质**：1080P以上
- **字幕**：必须加，很多用户在静音状态下浏览
- **BGM**：选择不影响说话的背景音乐
- **节奏**：剪掉所有停顿和废话，保持紧凑
- **封面**：单独设计封面，不要用视频截图

### 视频加载优化（2025新要求）

- 平台强化"视频加载速度"指标
- 确保视频加载时长低于2秒
- 避免超高码率导致加载缓慢

## 七、话题标签策略

### 话题标签的作用

- 增加内容在搜索结果中的曝光
- 帮助算法理解你的内容分类
- 让目标用户通过话题找到你

### 话题标签使用规范

- 数量：3-5个为最佳
- 选择：1个大话题 + 2个精准话题 + 1-2个长尾话题
- 优先使用官方推荐话题和高流量话题
- 避免堆砌不相关的热门话题（算法会判定为作弊）

### 话题标签示例

假设你做穿搭内容：
- 大话题：#穿搭
- 精准话题：#秋冬穿搭 #小个子穿搭
- 长尾话题：#梨形身材穿搭 #通勤穿搭分享

## 八、笔记转写/改写技巧

### 什么是笔记转写

把其他形式的内容（视频、文章、口述）转化为小红书笔记格式。

### 转写步骤

1. **提取核心信息**：从原始内容中提取3-5个核心观点
2. **重构标题**：按标题公式重写，融入搜索关键词
3. **适配正文格式**：短段落 + emoji + 分点 + 口语化
4. **补充互动元素**：添加提问、投票、引导收藏
5. **设计封面**：根据内容选择合适的封面形式

### 改写注意事项

- 不能照搬原文，必须用自己的语言重新表达
- 原创性权重已提升至35%，搬运必被限流
- 加入个人观点和经验，增加原创度
- 适配小红书的口语化风格，避免书面语
