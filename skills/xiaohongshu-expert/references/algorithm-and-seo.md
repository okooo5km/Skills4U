## 一、推荐算法深度拆解

### 笔记发布后的生命周期

```
发布 → 机器审核 → [通过] → 收录 → 匹配推送 → 初始流量池(200-500人)
                    ↓                              ↓
                 [不通过]                    30分钟内CTR+CES评估
                    ↓                              ↓
                 限流/删除              [数据好] → 更大流量池 → 持续晋级
                                        [数据差] → 停止推荐
```

### 审核阶段

**三层审核**：
1. 机器审核（秒级）：检查违禁词、敏感图片、重复内容
2. 人工审核（分钟-小时级）：机器不确定的内容转人工
3. 复审（随时触发）：被举报或达到某个曝光量级后可能触发

**审核通过的信号**：
用其他账号搜索你的笔记标题，能在"最新"排序中找到 = 已收录

### 收录阶段

- 只有被收录的笔记才能被推荐和搜索到
- 收录 ≠ 推荐，收录只是进入候选池
- 检查方法：搜索笔记标题 → 点"最新" → 能找到就是已收录

### 匹配推送阶段

系统通过"召回→粗排→精排→重排"四层漏斗筛选内容：

1. **召回**：从海量内容中初步筛选可能感兴趣的内容
2. **粗排**：快速排序，过滤低质内容
3. **精排**：精细排序，考虑内容质量、用户匹配度
4. **重排**：最终调整，减少同质化内容（打破信息茧房）

### 2025算法重大变化

- **原创性权重提升至35%**：抄袭/搬运/低质拼接必死
- **用户心智停留度**：截图 > 10次点赞（深度参与 > 浅层互动）
- **信息密度要求**：图文需有实质内容，不接受"封面+关键词堆砌"
- **视频完播率权重增加**：短视频也要有实质内容
- **去中心化加强**：超50%流量分配给1000粉以下用户

## 二、四大流量入口

### 1."发现"页面（最大流量入口）

- 打开小红书默认显示的页面
- 双列信息流展示
- 用户根据封面+标题决定是否点击
- **关键优化点**：封面质量、标题吸引力

### 2."搜索"页面（增长最快的入口，占50-65%）

- 用户主动搜索关键词
- 搜索流量 = 精准流量 = 高转化流量
- 日均搜索量6亿次
- **关键优化点**：关键词布局、标题SEO、内容相关性

### 3."关注"页面

- 展示已关注博主的内容
- 粉丝越多，这个入口的流量越大
- **关键优化点**：粉丝粘性、更新频率

### 4."同城"页面

- 基于地理位置的内容推荐
- 适合本地商家、探店博主
- **关键优化点**：发布时添加地理位置标签

## 三、CES评分机制

### 评分公式

```
CES = 点赞×1 + 收藏×1 + 评论×4 + 转发×4 + 关注×8
```

### 权重解读

| 行为 | 权重 | 含义 |
|---|---|---|
| 关注 | ×8 | 最高权重，代表用户长期价值 |
| 评论 | ×4 | 高权重，代表深度互动 |
| 转发 | ×4 | 高权重，代表社交传播力 |
| 点赞 | ×1 | 基础互动 |
| 收藏 | ×1 | 代表内容实用价值 |

### 实操启示

- **评论 > 点赞**：在正文中设置提问，引导评论
- **关注权重最高**：在笔记中引导关注，如"关注我持续分享xx"
- **转发价值高**：创作用户愿意分享给朋友的内容（实用干货/有趣/有共鸣）
- **收藏 = 实用**：干货类内容收藏率高，利于长尾流量

### 时间衰减因子

- 互动数据乘以衰减系数（λ≈0.1~0.3）
- **黄金2小时**：发布后2小时内的互动权重最高
- 越早期的互动数据，对推荐的影响越大
- 发布后立即在评论区互动，带动初始数据

## 四、搜索SEO优化策略

### 为什么SEO是2025年最重要的运营策略

- 搜索流量占比50-65%（每100个曝光中65个来自搜索）
- 搜索流量 = 精准流量，转化率远高于推荐流量
- 搜索页排名稳定，一篇笔记能吃半年流量
- 搜索用户信任度高，点击率30-80%（vs普通广告2-5%）

### 关键词研究方法

**1. 搜索框联想词**
- 在小红书搜索框输入核心词，记录所有下拉联想词
- 这些词 = 用户真实搜索需求

**2. TF-IDF策略**
- 筛选"高搜索量+低竞争度"的长尾关键词
- 工具：千瓜数据、新红数据、5118

**3. 竞品关键词分析**
- 分析同赛道爆款笔记使用了哪些关键词
- 找到竞品覆盖不到的长尾词

### 关键词布局实操

**标题**：
- 前10个字必须包含核心关键词（权重占60%+）
- 前18字必须包含2个核心搜索词
- 示例：「小个子穿搭｜155cm秋冬显高搭配，这5套太绝了」
  - 核心词："小个子穿搭"在前6字
  - 第二核心词："秋冬显高搭配"在前18字内

**正文**：
- 前200字自然嵌入长尾关键词
- 每300字重复1次目标关键词
- 不要堆砌，要自然融入语境

**话题标签**：
- 3-5个相关标签
- 优先使用官方话题和高流量话题

**图片ALT信息**：
- 上传图片时添加关键词相关描述

## 五、搜索排名机制

### 排名四维模型

```
排名 = 内容质量分(40%) + 用户互动分(30%) + 账号权重分(20%) + 实时热度分(10%)
```

### 各维度详解

**内容质量分（40%）**：
- 原创度
- 信息密度
- 与搜索关键词的相关性
- 内容完整度

**用户互动分（30%）**：
- CES评分（评论和转发权重最高）
- 互动率（互动数/曝光数）
- 互动质量（真实评论 > 水评论）

**账号权重分（20%）**：
- 账号垂直度：持续发布同一领域内容
- 账号活跃度：定期更新、回复评论
- 粉丝量级和粉丝质量
- 账号年龄和历史表现

**实时热度分（10%）**：
- 近期互动数据的增速
- 是否与当前热点相关

### 提升搜索排名的实操方法

1. **深耕垂直领域**：持续发布同类内容提升账号权重
2. **优化关键词布局**：标题+正文+标签全方位覆盖
3. **提升互动数据**：设置互动引导，鼓励评论和转发
4. **定期更新旧笔记**：补充最新信息、更新关键词
5. **发起粉丝投票/直播互动**：提升账号活跃度

## 六、限流原因与排查

### 限流的表现

- 笔记发布后曝光量异常低（小眼睛<100）
- 笔记只推荐给粉丝，不推荐给新用户
- 搜索自己的笔记标题找不到
- 账号整体数据持续下滑

### 限流的6大原因

**1. 内容违规**
- 含有敏感词/违禁词
- 含有联系方式/外部链接/二维码
- 涉及虚假宣传
- 排查方法：检查笔记中是否有平台禁用词

**2. 搬运/抄袭**
- 非原创内容
- 图片被识别为搬运
- 文案与已有笔记高度重复
- 排查方法：确保所有内容为原创

**3. 低质AI内容**
- 2025年处置了60万篇低质AI笔记
- 纯AI生成未经人工润色的内容
- 未标注AI生成标识
- 排查方法：AI辅助可以，但必须人工润色并标注

**4. 数据异常**
- 刷赞/刷评论/刷粉丝
- 互粉互赞群组行为
- 排查方法：确保所有互动数据真实

**5. 广告嫌疑**
- 硬植入广告内容
- 未通过蒲公英平台的商业合作
- 笔记中出现品牌官方标志
- 排查方法：商业内容走蒲公英平台

**6. 隐性限流**
- 无明显违规但流量低
- 可能原因：内容质量不高、选题竞争激烈、发布时间不佳
- 排查方法：优化内容质量、更换选题、调整发布时间

### 限流应对方案

1. 检查并删除违规内容
2. 修改后重新发布（不要在原笔记上修改）
3. 连续发布3-5篇高质量原创内容"养号"
4. 保持正常互动频率
5. 如果账号整体被限流，暂停发布3-7天后恢复
