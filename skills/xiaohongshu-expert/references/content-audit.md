## 一、发布前内容审核（Pre-publish Audit）

当用户请求审核一篇准备发布的笔记时，按以下维度逐项检查并给出诊断报告。

### 审核流程

要求用户提供：笔记标题、正文内容、配图描述（如有）、话题标签。然后按下面5个维度出具审核报告。

### 维度1：合规安全检查 🔴

这是最高优先级。任何一项不通过，建议修改后再发布。

**检查项**：

| 检查项 | 风险等级 | 说明 |
|---|---|---|
| 敏感词/违禁词 | 🔴 致命 | 参见本文第四节敏感词库，逐一比对 |
| 联系方式/外链 | 🔴 致命 | 微信号、QQ、手机号、二维码、外部链接、暗号引导 |
| 虚假宣传 | 🔴 致命 | "最好""第一""100%有效"等绝对化用语 |
| 医疗健康承诺 | 🔴 致命 | 疗效承诺、诊断建议、处方药推荐 |
| 虚假人设 | 🔴 致命 | 伪造学历/收入/身份/经历 |
| AI内容标注 | 🟡 高危 | 如使用AI辅助创作，是否已标注 |
| 版权风险 | 🟡 高危 | 图片/文字/音乐是否存在侵权风险 |
| 其他平台水印 | 🟡 高危 | 图片/视频是否带有抖音、快手等水印 |

### 维度2：SEO优化检查 🟢

**检查项**：

| 检查项 | 标准 | 诊断方法 |
|---|---|---|
| 标题关键词 | 前10字包含核心关键词 | 提取标题前10字，检查是否含搜索热词 |
| 标题双关键词 | 前18字包含2个核心词 | 检查前18字的关键词覆盖 |
| 标题长度 | ≤20字 | 字数统计 |
| 正文关键词密度 | 每300字出现1次目标词 | 统计关键词出现频率 |
| 前200字布局 | 前200字包含长尾关键词 | 检查开头部分 |
| 话题标签 | 3-5个相关标签 | 检查标签数量和相关性 |
| 关键词堆砌 | 不能堆砌 | 如果同一关键词在短段落中出现3次以上，判定为堆砌 |

### 维度3：内容质量检查 🟢

**检查项**：

| 检查项 | 标准 | 说明 |
|---|---|---|
| 信息密度 | 有实质性内容 | 不能只有封面+关键词，必须有干货 |
| 原创度 | 原创表达 | 不能照搬其他笔记的句式和结构 |
| 正文结构 | 开头+中间+结尾 | 是否有清晰的内容结构 |
| 段落长度 | 每段≤3行 | 适合手机阅读 |
| 排版可读性 | emoji分段+短段落 | 是否有良好的视觉层次 |
| 图片数量 | 6-9张（图文笔记） | 图片是否充足 |
| 视频时长 | 1-3分钟 | 是否在最佳时长范围 |

### 维度4：互动潜力评估 🟢

**检查项**：

| 检查项 | 标准 | 说明 |
|---|---|---|
| 标题吸引力 | 是否激发好奇/痛点/反差 | 用户会不会想点进来 |
| 互动引导 | 结尾是否设置提问/投票 | CES评分中评论权重×4 |
| 收藏动机 | 内容是否有"收藏价值" | 干货类/教程类/合集类收藏率高 |
| 转发动机 | 内容是否让人想分享 | 有共鸣/有用/有趣 |
| 争议性 | 是否能引发讨论 | 适度的争议 = 评论区活跃 |

### 维度5：发布策略建议 🟢

根据内容类型，给出：
- 推荐发布时间段
- 发布后黄金2小时的互动策略
- 评论区首条评论建议（自己写什么）
- 预期数据范围（基于内容类型和赛道）

### 审核报告输出格式

```
📋 笔记发布前审核报告
━━━━━━━━━━━━━━━━━━

✅/❌ 合规安全：[通过/不通过] — [具体问题]
✅/⚠️ SEO优化：[评分/10] — [具体建议]
✅/⚠️ 内容质量：[评分/10] — [具体建议]
✅/⚠️ 互动潜力：[评分/10] — [具体建议]
📅 发布建议：[推荐时间] + [互动策略]

🔧 修改建议：
1. [最高优先级修改]
2. [次优先级修改]
3. ...

📊 综合评分：X/10
💡 预判：[对这篇笔记表现的预期判断]
```

## 二、账号健康度诊断（Account Health Check）

当用户怀疑账号有问题或想评估账号状态时，执行全面诊断。

### 信息收集

先向用户收集以下信息（能提供多少算多少）：

**基础信息**：
- 账号粉丝数
- 账号运营多久了
- 最近30天发了多少篇笔记
- 账号所在赛道/定位

**数据表现**：
- 最近10篇笔记的平均小眼睛（曝光量）
- 最近10篇笔记的平均互动量（点赞+收藏+评论）
- 最近一个月的粉丝增减情况
- 是否有某篇笔记数据突然异常

**行为历史**：
- 是否收到过平台警告/通知
- 是否有笔记被删除/限流
- 是否使用过互赞群/刷粉服务
- 是否有过站外引流行为
- 是否频繁修改/删除笔记

### 诊断四维模型

**维度1：账号权重评估**

| 指标 | 健康 | 亚健康 | 危险 |
|---|---|---|---|
| 笔记平均曝光 | 稳定或增长 | 波动大 | 持续下降 |
| 粉丝增长 | 正增长 | 停滞 | 负增长 |
| 互动率 | ≥3% | 1-3% | <1% |
| 收录率 | ≥90% | 70-90% | <70% |

**维度2：内容健康度**

| 指标 | 健康 | 亚健康 | 危险 |
|---|---|---|---|
| 垂直度 | ≥80%同领域 | 50-80% | <50%杂乱 |
| 更新频率 | 每周≥3篇 | 每周1-2篇 | 不规律/断更 |
| 原创度 | 100%原创 | 部分借鉴 | 搬运/洗稿 |
| 内容质量 | 有信息密度 | 一般 | 低质/水贴 |

**维度3：行为合规度**

| 指标 | 健康 | 亚健康 | 危险 |
|---|---|---|---|
| 违规记录 | 0次 | 1-2次轻微 | 3次以上/严重违规 |
| 引流行为 | 无 | 曾有但已停止 | 正在进行 |
| 数据真实性 | 100%真实 | 曾刷过但已停 | 正在刷量 |
| 频繁操作 | 正常 | 偶尔批量删帖 | 频繁异常操作 |

**维度4：变现健康度**（仅适用于已变现的账号）

| 指标 | 健康 | 亚健康 | 危险 |
|---|---|---|---|
| 广告比例 | ≤30%笔记 | 30-50% | >50%全是广告 |
| 蒲公英报备 | 100%报备 | 部分报备 | 不报备 |
| 粉丝信任 | 互动正常 | 互动下降 | 大量取关/负面评论 |

### 健康度评分体系

```
账号健康度 = 权重评估(30%) + 内容健康(30%) + 行为合规(25%) + 变现健康(15%)
```

**评分等级**：
- 90-100：💚 优秀 — 继续保持
- 70-89：💛 良好 — 有小问题需注意
- 50-69：🟠 亚健康 — 需要重点调整
- 30-49：🔴 危险 — 需要立即整改
- 0-29：⛔ 重症 — 考虑账号恢复或重建

### 诊断报告输出格式

```
🏥 账号健康度诊断报告
━━━━━━━━━━━━━━━━━━

📊 综合健康评分：XX/100 [等级emoji]

维度明细：
  账号权重：XX/100 — [简要说明]
  内容健康：XX/100 — [简要说明]
  行为合规：XX/100 — [简要说明]
  变现健康：XX/100 — [简要说明]（如适用）

🔍 发现的问题：
  🔴 [严重问题1]
  🟡 [中等问题1]
  💡 [优化建议1]

💊 恢复/优化方案：
  立即执行：
    1. [最紧急的操作]
    2. [次紧急的操作]
  
  7天计划：
    Day 1-3: [具体操作]
    Day 4-7: [具体操作]
  
  30天计划：
    [长期调整方向]

📈 预期恢复时间线：[估计]
```

## 三、限流深度排查与恢复方案

### 第一步：确认是否真的被限流

**不一定是限流的情况**：
- 新账号本来就没有流量基础
- 内容选题竞争激烈，被更优质的内容压制
- 发布时间不佳
- 标题/封面不吸引人

**确认限流的方法**：
1. 用另一个账号搜索你的笔记标题 → 在"最新"中找不到 = 未收录（可能限流）
2. 连续5篇笔记小眼睛<100 = 大概率限流
3. 收到平台违规通知 = 确认限流
4. 笔记只有粉丝互动、没有新用户发现 = 疑似限流

### 第二步：定位限流原因

**原因排查优先级**（从最常见到最少见）：

1. **内容违规**（60%的限流原因）
   - 检查所有近期笔记的敏感词
   - 检查图片中是否有违规信息
   - 检查是否有未报备的商业内容

2. **AI低质内容**（2025年新增高频原因）
   - 纯AI生成未润色
   - 批量发布AI水文
   - 未标注AI生成

3. **搬运/抄袭**
   - 图片搬运（即使改了文字，图被识别也算）
   - 文案洗稿（换了几个词但结构一样）

4. **行为异常**
   - 互赞群/刷量
   - 短期内大量删帖
   - 频繁切换设备

5. **账号历史问题**
   - 之前的违规积累到一定程度触发限流
   - 账号被举报

### 第三步：对症恢复方案

**情况A：单篇笔记限流**
1. 不要修改该笔记（修改后再审核更严格）
2. 直接删除问题笔记
3. 24小时后发布新的高质量原创内容
4. 观察新笔记的数据表现

**情况B：账号级别限流（轻度）**
1. 暂停发布3天
2. 期间正常浏览、点赞、评论（模拟正常用户行为）
3. 第4天开始每天发1篇高质量原创内容
4. 连续7天后评估数据是否恢复

**情况C：账号级别限流（重度）**
1. 暂停发布7天
2. 彻底清理所有违规内容（删除问题笔记）
3. 检查并修改账号信息（昵称、简介中是否有违规词）
4. 第8天起每天发1篇高质量原创内容
5. 连续14天后评估
6. 如仍无改善，尝试通过"帮助与反馈"向平台申诉
7. 30天仍无改善，建议评估是否需要新号重新开始

**情况D：被封号/严重处罚**
1. 通过官方渠道申诉
2. 提供相关证据说明
3. 如申诉失败，只能新号重来
4. 新号注意：不要用同一设备/手机号/IP

## 四、敏感词与违规词库

### 绝对禁止词（触发即删帖/限流）

**引流类**：
微信、wx、VX、薇信、V信、QQ、扣扣、私聊加我、关注公众号、淘宝搜、拼多多、抖音看、快手搜

**虚假宣传类**：
最好、最强、第一、独一无二、100%有效、药到病除、立竿见影、无副作用、国家级、世界级

**医疗违规类**：
治愈、根治、特效药、偏方治百病、诊断、处方、替代就医

**金融违规类**：
保证收益、稳赚不赔、日入X万、零风险投资

**政治敏感类**：
涉及国家领导人、政治事件等敏感话题

### 高风险词（可能触发审核/降权）

**软广嫌疑词**：
推荐、安利、种草、必买、回购、链接在、同款、同色号（当与品牌名密集出现时风险更高）

**擦边词**：
各类谐音词、暗号、缩写（平台持续更新识别能力）

### 规避方法

- 不要用谐音/缩写来代替敏感词（平台能识别）
- 商业推荐走蒲公英平台
- 如需表达"最好"类含义，用"我个人觉得""在我用过的里面"等个人体验视角
- 医疗/健康类内容加"个人经验分享，不构成医疗建议"免责声明

## 五、AI内容合规指南

### 法律背景

2025年9月1日起，《人工智能生成合成内容标识办法》正式生效。这是国家层面的法规，不是平台自己的规定。

### 小红书平台执行标准

- AI识别准确率约95%（机器识别+人工识别）
- 2025年上半年处置低质AI笔记60万篇
- 用户可通过"设置"主动声明"由AI生成"
- 对未声明的疑似AI内容，平台自动添加标识

### AI内容的正确使用方式

**可以做**：
- 用AI辅助选题灵感
- 用AI生成初稿，然后大幅人工改写
- 用AI辅助翻译、校对
- 用AI生成配图（标注"AI生成"）

**不能做**：
- 直接复制AI输出发布（未经润色）
- 批量用AI生成大量笔记
- AI生成内容不标注
- 用AI伪造个人经历/体验

### 安全操作建议

1. AI辅助创作后，至少改写60%以上的内容
2. 加入个人真实经验和观点
3. 主动在笔记中标注"部分内容由AI辅助生成"
4. 确保核心信息准确，AI可能产生幻觉

## 六、商业内容合规指南

### 2025年广告内容新要求

"小红书比用户更讨厌软广，因为软广破坏的是平台的信任根基。"

### 合规商业内容的标准

**必须通过蒲公英平台报备的内容**：
- 品牌方付费的产品推广
- 品牌置换合作
- 任何获取了商业利益的推荐内容

**不需要报备的内容**：
- 自购产品的真实分享（无商业利益）
- 个人日常推荐（无品牌合作关系）

### 商业内容创作规范

**做**：
- 保持真诚的使用体验
- 如实描述产品优缺点
- 通过蒲公英报备
- 标注"合作""推广"字样

**不做**：
- 硬植入广告
- 全文吹捧无缺点
- 虚假使用体验
- 未报备的商业推广
- 在笔记中直接出现品牌官方素材/标志

### 广告比例健康线

- 建议广告内容占比≤30%
- 每发1篇广告，至少发2-3篇非商业内容
- 保持粉丝对账号的信任度
